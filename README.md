## Subindo container com banco de dados

```
$ docker-compose up -d
```

## Instalação

```bash
$ npm install
```

## Criando tabelas no banco de dados

```bash 
$ npm run init:db
```

## Rodando aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
